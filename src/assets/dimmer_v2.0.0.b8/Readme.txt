Dimmer

Version: 2.0.0 b8
Release date: 31 Jul 2020
Developed by: Nelson Pires
Website: https://www.nelsonpires.com/software/dimmer

Description:
------------------------
The original eye saver app designed to provide brightness controls for
LCD/TFT/LED screens. Useful for working in low light conditions where
too much screen brightness is undesirable and bad for the eyes.
Saving the eyesight and eye strain of thousands of users since 2004.

It's a portable app:
 - runs without installation
 - un-installs by deleting the files

NOTE: If the program fails to create the json config file, you are in a
privileged/protected folder. Run as Administrator or from another folder.

Beta software:
------------------------
Please be advised, this software is currently in beta version.
Do not use in mission critical environments.

For more information please visit the official website.
